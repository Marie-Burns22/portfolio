Technical Project Introduction

My technical projects are inspired by friends, family, and students. In each project I identify what information the user needs and then design a web application to collect and organize that data.

Each project was either designed to utilize a technology I was learning or was designed with the technologies that best suited the purpose of the application.

Specific technologies used for each project are listed in the project descriptions below. The main stack I rely on is Ruby on Rails for the backend and Javascript, React, and Redux for the frontend.
